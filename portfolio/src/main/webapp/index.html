My Work

6.08 Introduction to Embedded Systems

During the spring semester of my first year at MIT, I enrolled in 6.08 Introduction to Embedded Systems. In this class, I programmed an ESP32 microcontroller in C++ using Arduino libraries. Throughout the class, multiple input and output devices were added to this microcontroller, including a GPS, a gyroscope and an LCD display. I also worked with a flask-based server backend and a SQLite database. This class culminated in a four-week capstone project utilizing the server backend and the ESP32. My team created a virtual world where the user can create an avatar and have it be displayed on the ESP32's display. In addition, players were able to invite up to four other users to their room and view multiple avatars on the screen simultaneously. My main focuses in this project included an authentication system that stored usernames and hashed password data, and integrating the ESP32 frontend and server backend to allow for near real-time updates to each avatar's appearance, animation and location.
